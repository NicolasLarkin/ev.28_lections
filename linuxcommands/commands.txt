Linux commands
Ctrl  + alt + T -  команда открытия Терминала
~ - символ домашней директории 
/ - символ корневой директории

 . - обозначает текущую директорию
 .. - обозначает предыдущую директорию
 
1. ls - команда для вывода всех директорий(папка) и файлов в текущей(рабоче) дирекории в терминале.

2. cd (change directory) - команда для перехода с одной директории в другую. 

3. Mkdir - создание директории

4. Touch - создает файл 

5. Rmdir - удаляет пустую директорию

6. rm - удаляет файл

7. rm -rf - удаляет вообще все (принудительное удаление) (recursively force)		

8. mv - команда для переименования и перемещения файла (move)
	mv <название> <новое название>
	mv <название> <адрес/> 
	
9. cp - (copy) копирование
	cp <Имя> <куда>

10. cat - открытие данных файла без открытия визуалки, содержания файла

11. nano - открытие визуалкии для редактирования 
	nano если нет файла то оно создаст новый

12. echo - добавить запись в файл без открытия визуалки  (быстрая запись текста в файл)
	echo текст > файл

13 ls -a - показывает скрытые и обычные файлы и директории

14 ls -la - показывает подробную информацию (list all)

15 pwd - показывает весь путь ( print working directory) выводит всеь путь директории (текущей)

16 ctrl + L чистка терминала
